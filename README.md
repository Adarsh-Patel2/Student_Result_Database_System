# Student Result Management System

A web-based application for managing student records, marks, and result generation.

##  Features

- Add Student Details
- View Student Records
- Generate Student Results
- Display Student Marks
- User-Friendly Interface

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Java
- JSP
- Spring Boot

### Database
- MySQL
